# Todo App 使い方ガイド

シンプルで使いやすいTodoアプリです。タスク管理を簡単に行えます。

## 🚀 アプリの起動方法

### 方法1: ブラウザで直接開く（最も簡単）

1. Finderでプロジェクトフォルダを開く
2. `index.html` ファイルを見つける
3. `index.html` を **ダブルクリック**
4. デフォルトのブラウザでアプリが開きます

または

1. `index.html` ファイルを **右クリック**
2. 「このアプリケーションで開く」→ 使いたいブラウザ（Chrome、Safari、Firefoxなど）を選択

### 方法2: ローカルサーバーを使う（推奨）

ローカルサーバーを使うとより本格的な動作確認ができます。

#### Pythonを使う場合（Macに標準搭載）

```bash
# プロジェクトフォルダに移動
cd /path/to/ClaudeProjects

# Python 3でサーバー起動
python3 -m http.server 8000
```

その後、ブラウザで以下のURLを開く:
```
http://localhost:8000
```

サーバーを停止するには、ターミナルで `Ctrl + C` を押します。

#### Node.jsのhttp-serverを使う場合

```bash
# http-serverをインストール（初回のみ）
npm install -g http-server

# プロジェクトフォルダに移動
cd /path/to/ClaudeProjects

# サーバー起動
http-server -p 8000
```

ブラウザで `http://localhost:8000` を開きます。

#### VS Codeの拡張機能を使う場合

1. VS Codeで「Live Server」拡張機能をインストール
2. `index.html` を右クリック
3. 「Open with Live Server」を選択
4. 自動的にブラウザが開きます

## 📖 機能の使い方

### 1. タスクを追加する

1. 画面上部の入力欄に新しいタスクを入力
2. 「追加」ボタンをクリック、または **Enterキー** を押す
3. タスクがリストに追加されます

**例:** 「買い物に行く」「レポートを書く」など

### 2. タスクを完了にする

1. タスク左側の **チェックボックス** をクリック
2. タスクに取り消し線が引かれ、半透明になります
3. もう一度クリックすると未完了に戻ります

### 3. タスクを編集する

1. 編集したいタスクの **「編集」ボタン**（緑色）をクリック
2. ポップアップが表示されるので、新しい内容を入力
3. OKをクリックして確定

### 4. タスクを削除する

1. 削除したいタスクの **「削除」ボタン**（赤色）をクリック
2. 確認ダイアログが表示されます
3. OKをクリックすると削除されます

### 5. タスクをフィルタリングする

画面中央の3つのボタンでタスクを絞り込めます:

- **すべて**: 全てのタスクを表示
- **未完了**: 完了していないタスクのみ表示
- **完了済み**: 完了したタスクのみ表示

### 6. 統計を確認する

画面下部に以下の情報が表示されます:

- **総タスク数**: 登録されている全タスク数
- **完了**: 完了済みのタスク数
- **未完了**: まだ完了していないタスク数

## 💾 データの保存について

- タスクは **ブラウザのlocalStorage** に自動保存されます
- ブラウザを閉じても **データは消えません**
- 同じブラウザで開けば、前回のタスクが表示されます

### 注意点

- **異なるブラウザ**では別々のデータになります
  - 例: Chromeで追加したタスクはSafariでは表示されません
- **プライベートブラウジング（シークレットモード）**ではデータが保存されません
- ブラウザのキャッシュをクリアするとデータが消えます

## 🔧 トラブルシューティング

### タスクが表示されない

- ページを再読み込み（F5キーまたはCmd+R）してみてください
- ブラウザのコンソールでエラーが出ていないか確認してください

### スタイルが崩れている

- `index.html`、`style.css`、`app.js` の3つのファイルが同じフォルダにあるか確認してください
- ブラウザのキャッシュをクリアしてみてください

### データが保存されない

- プライベートブラウジングモードを使用していないか確認してください
- ブラウザのlocalStorageが有効になっているか確認してください

## 📱 スマートフォンでの使用

このアプリはレスポンシブデザインなので、スマートフォンでも使えます:

1. ローカルサーバーを起動
2. 同じWi-Fi内のスマートフォンから、PCのIPアドレスとポート番号でアクセス

例: `http://192.168.1.10:8000`
